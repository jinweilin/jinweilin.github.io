ISO/IEC 27701 落實建議方案

專案目標：
打造兼具合規性、使用者便利性與可追蹤性的 PII（個人識別資訊）資料控管機制，符合 ISO/IEC 27701 國際隱私管理標準，強化公司內部資安治理。

1️⃣ 磁碟加密處理管理

目的： 防止設備遺失造成 PII 外洩

項目解決方案工具/平台成本預估
磁碟全盤加密強制公司筆電開機前加密BitLocker / FileVault無額外成本
密鑰管理使用 TPM/BIOS 密碼搭配集中備份Azure AD Recovery Key需 E3+（可選）
稽核紀錄建立裝置清單 + 半年查核Excel / Power BI無

備註：筆電全面加密，桌機依敏感等級選擇性加密。

2️⃣ PII 檔案掃描與自動刪除

目的： 避免檔案長期堆積，違反最小保存原則

項目解決方案工具成本預估
自動掃描使用正則表達式掃描個資PowerShell / Python內部開發
自動刪除180 天未存取檔自動清除任務排程無
稽核紀錄Log 刪除 + Email 給 DPOSMTP + CSV無

3️⃣ Email (PST) 管理

目的： 控管 Outlook 中長期保存的個資風險

項目解決方案工具成本預估
PST 掃描讀取並過濾含個資信件Python + pypff開源
自動刪除180 天未讀取信件刪除Task Scheduler無
稽核紀錄產出 CSV + 報表腳本無

4️⃣ NAS 管理建議

目的： 避免 NAS 長期堆積 PII

項目解決方案工具成本預估
掃描工具部署掃描腳本Python + Synology Task免費
自動搬移/刪除搬移至 quarantine 後自動清除腳本內部開發
稽核紀錄建立清理紀錄表Google Sheet / Power BI無

5️⃣ 辦公空間門禁強化

區域解決方案工具成本預估
門禁強化RFID + 人臉辨識HID / 雲端廠商5~10 萬 NTD
訪客管理QR Code + 登記Google 表單5,000 NTD 以下
監控記錄門禁 API 導出Excel 匯出無
